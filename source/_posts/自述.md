vscode商店地址 [戳这里](https://marketplace.visualstudio.com/items?itemName=zhucy.project-tree)  
example

```
project-tree
...
├─ src
│  ├─ config.ts
│  ├─ extension.ts
│  ├─ ignore
│  │  └─ index.ts
│  ├─ ignore.ts
│  ├─ index.ts
│  ├─ test
│  │  ├─ extension.test.ts
│  │  └─ index.ts

├─ tsconfig.json
├─ tslint.json
└─ webpack.config.js    
```
